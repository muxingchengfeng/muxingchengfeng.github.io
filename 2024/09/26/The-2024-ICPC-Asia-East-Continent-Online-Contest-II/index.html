The 2024 ICPC Asia East Continent Online Contest (II)

The 2024 ICPC Asia East Continent Online Contest (II)（A、C、E、F、G、H、I、J、K、L）

A - Gambling on Choosing Regionals

最坏情况下强队和自己都去同一站，此时选取最小的赛站， $rank$ 越小。按照能力值排序即可。

F - Tourist

按照题意模拟即可。

G - Game

若 $V_{Alice} >= V_{Bob}$ ，则可以让 $Bob$ 先赢下 $0 - (V_{Alice} / V_{Bob} - 1)$ 次，此时 $Alice$ 赢一次 ，依然可以保证赢下比赛。

所以赢得概率为 $p_0 + p_0p_1 + …+ p_0p_1^{(V_{Alice} / V_{Bob} - 1)}$ ，可以使用等比数列求和。

若 $Bob$ 赢了 $V_{Alice} / V_{Bob}$ ，则 $V_{Alice} = V_{Alice} % V_{Bob}$ 。则 $V_{Alice} < V_{Bob}$。

若若 $V_{Alice} < V_{Bob}$ ，则 $Alice$ 必须先赢下 $V_{Bob} / V_{Alice}$ 次，此时 $V_{Bob} = V_{Bob} % V_{Alice}$ 赢一次 。则 $V_{Alice} >= V_{Bob}$ 。

I - Strange Binary

对于二进制最低位两位都是 $0$ ，则无解。

对于二进制一位为 $1$ ，如果他的高一位也是 $1$ ，那么答案中设为 $1$，如果高一位是 $0$ ，即原二进制中为 $01$，那么答案中则可以设置为上一位减去下一位，即 $1, -1$ ，同时注意到若更高一位也为 $0$ ，即 $001$ ，那么可以设置为 $1, -1, -1$ ，以此类推。

J - Stacking of Goods

若第 $i$ 个此时在第 $j$ 个上方，产生的贡献是 $v_i - c_i * W + v _j - c_j * (W + w_i)$ ，若第 $i$ 个在第 $j$ 个下方，产生的贡献为 $v_i-c_i * (W + w_j) + v_j-c_j*W$ 。此时贡献越小越好，化简得若 $w_i/c_i>w_j/c_j$ ，则第 $i$ 个在第 $j$ 个上方，反正在下方。排序即可。

k - Match

对于形如 $a \bigoplus b >= k$ ，我们从高位向低位枚举。

$f(A, B, k)$ 表示 $A$ 集合和 $B$ 集合进行 $A_i \bigoplus B_j ≥ k$ 匹配的答案，返回一个数组，依次 表示 $0, 1, 2, 3,…, n$ 个匹配的方案数。

假设 $k$ 这一位的权值为 $0$，那么所有 $A_i$ , $B_j$ 只要当前位不同就可以进行匹配。 我们把 $A$ 按当前位为 $0$ 或 $1$ 分类成 $A_0$, $A_1$，$B$ 同理分成 $B_0$, $B_1$，先算出 $f(A_0, B_0, k)$, $f(A_1, B_1, k)$， 剩下的匹配在 $A_0$, $B_1$ 和 $B_1$, $A_0$ 中都可以任意匹配，可以组合数 DP 转移。转移如下：

我们令 $f(A_0, B_0, k) = f_0$ ，$f(A_1, B_1, k) = f_1$ ，那么 $f_0[i]$ 表示从 $A_0, B_0$ 里面分别取出 $i$ 个且匹配的合法方案数。$f_1[j]$ 同理。我们设从 $f_0$ 里面取 $i$ 个，$f_1$ 里面取 $j$ 个，同时我们从剩下的 $A_0$ 里面取出 $u$ 个，剩下的 $B_1$ 里面取出 $u$ 个并两两任意匹配，从剩下的 $A_1$ 里面取出 $v$ 个，剩下的 $B_0$ 里面取出 $v$ 个并两两任意匹配。那么匹配方案有：

$dp[i+j+u+v] = f_0[i] * f_1[j] * C(cnta_0 - i, u) * C(cntb_1-j, u) * u!* C(cnta_1 - j, v) * C(cntb_0 - i, v) * v!$ 。

直接四重枚举即可，我们可以发现这类似于树上背包。复杂度 $O(n^4)$ 。

假设 $k$ 这一位的权值为 $1$，那么只有 $f(A_1, B_0, k)$ 和 $f(A_0, B_1, k)$ 有解，将二者最高位抹去继续递归即可。

时间复杂度为 $O(n^4 )$ 。

L - 502 Bad Gateway

最优策略一定是选择一个 $c\in[1,t]$ ，不停重置，直至重置到一个小于等于 $c$ 的数就直接减到 $0$ 。

设重置到小于等于 $c$ 的数的概率为 $p = c/t$ ，由几何分布可知重置到小于等于 $c$ 的期望步数为：$E = p + 2p(1-p) + 3p(1-p)^2 + …. = 1/p$ 。

到达小于等于 $c$ 后会先减去 $1$ ，之后就是每步减去 $1$ ，这部分的期望步数：$E = (c-1)/2$ 。

所以总期望步数为：$(c-1)/2 + t/c$ 。这是一个对勾函数，在 $sqrt(2t)$ 出取最小值。
